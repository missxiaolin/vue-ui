### 单元测试

- chai 单元测试
- karma （[karma]卡玛）是一个单元测试运行器，加载测试脚本，然后运行测试用例
- Mocha （[mocha]摩卡）是一个单元测试框架/库，他可以用来写测试用例
- Sion  （西农）是一个 spy/stub/mock 库，用以辅助测试
